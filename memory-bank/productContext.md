# Product Context

## What it is

A $15/month micro-SaaS that gives independent real estate agents a copy-paste JavaScript widget or REST API endpoint showing live neighborhood market stats on their personal websites.

## Target audience

11,000+ real estate agents in Collin County (McKinney, Frisco, Plano, Allen, Prosper) and broader DFW metroplex.

## Core value prop

Agents need to prove local expertise on their websites but have no affordable, automated way to display live stats (median price, days on market, active inventory). This replaces expensive all-in-one platforms with a simple embed.

## Pricing

- $15/month flat (Lemon Squeezy, Merchant of Record — handles sales tax)

## Data

- Source: Redfin public data (S3, free, weekly cadence)
- Coverage: 17 zip codes across Collin County
- Metrics: median list price, median sale price, active listings, days on market
- Attribution required: "Powered by Redfin Data"

## Key constraints

- Data updates weekly (not daily as originally in ROADMAP — Redfin publishes monthly)
- Redfin ToU: must attribute, cannot redistribute raw data as competing product
- Using data for agent analytics widgets is in the acceptable use zone

## Future upsell potential

Redfin data also includes: median_sale_price, homes_sold, months_of_supply, avg_sale_to_list, sold_above_list, price_drops. Good candidates for a higher-tier plan.
